About {{ project_name }}

Project: {{ project_name }}

This project is a comprehensive smart home security system centered around a Raspberry Pi. It integrates various sensors for environmental and security monitoring, utilizing an Ultrasonic sensor to detect doorstep presence, triggering a full alert system with a buzzer, LED, and camera snapshot functionality. All data and control commands are managed through this web dashboard, which communicates with the Raspberry Pi hardware via Adafruit IO.

Hardware & Components

  • Raspberry Pi: (GPIO + I²C enabled)
  • Ultrasonic Sensor (HC-SR04):
    • TRIG → BCM 5
    • ECHO → BCM 12 (Use safe 3.3 V echo or voltage divider)
  • Camera: PiCamera2 (Picamera2)
  • I²C LCD 16x2: (PCF8574 @ 0x27 or 0x3F) — SDA/SCL on bus 1
  • Buzzer (Active): → BCM 18 (with series resistor)
  • Alert LED (Red): → BCM 21 (with series resistor)
  • Extra LEDs: Yellow BCM 16, Red BCM 23, Green BCM 24
  • Fan / Relay Module:
    • Fan Control → BCM 22
    • Relay IN (Active-Low) → BCM 17

*The system uses an Adafruit IO dashboard for data communication.*

Project Reflection

Milestone 2: Integrated System Deployment

In this milestone, we successfully achieved our main goal of integrating at least three devices and three sensors, and placing the entire system into a more organized enclosure. The addition of the LCD screen significantly enhanced the user experience by providing easily accessible, local status information.

We improved the detection system by switching from a passive motion sensor to an Ultrasonic sensor (HC-SR04), which allows for more reliable doorstep presence detection. This sensor is fully integrated with the buzzer, LCD, an LED, and the camera, creating a complete and responsive alert system.

However, the project presented some challenges. The complexity of the breadboard wiring, due to the high number of connected components, made modifications and debugging difficult. Furthermore, working with the relay module was a new experience, requiring extra time to correctly understand its wiring and operational principles. Our only remaining issue is that we were unable to achieve the desired speed control for the motor.

Final Part Checkpoint

Looking back at the whole project, We are really proud of how much we were able to integrate and how cohesive the system feels now. The biggest win was definitely moving from just a pile of components to a working, integrated security system managed through a clean web dashboard. We learned a ton about both hardware interfacing and web service integration.
